Program Quality Testing

Course: Advance Programming

Group project

Analyzed the corresponding for multiple speech processing tasks by implementing a speaker identification application, SpeakerIdentApp of MARF, to identify the speaker’s identities, gender and accent via machine learning for purpose of detecting the best candidates of the algorithm configurations. The Modular Audio Recognition Framework (MARF) is an open-source system written in Java to provide the testing framework to verify and validate algorithms of pattern recognition pipeline. MARF and GIPSY has been used to test comparing and analyzed same codes respectively. Then we feed backed the several different data, such as average cyclomatic complexity[v(G)], essential complexity[ev(G)] , module design complexity[iv(G)], etc. Then we exported the data by both textual and diagram forms. The three kinds of diagram results (Scatterplot, Histogram, and Kiviat) can state the difference between two test results obviously and directly. I was responsible for the data importing and result analysing.

Technologies: Java, MARF, GIPSY
